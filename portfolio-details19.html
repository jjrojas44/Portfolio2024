The UI for the terminator: resistance sequel,Terminator: Future war.

Case Study:

Terminator: Future war
Project Overview
  • Project Name: Terminator: Future war (Hypothetical Sequel)
  • Role: UX/UI Designer (Solo Designer)
  • Tools Used: Figma,Photoshop,Photopea,Gameuidatabase,dribbble,Behance,
  • Timeline: 3 Weeks
  • Objective: Introduce a hypothetical UI concept for a sequel called Terminator: Future War.
Value Added

The Terminator: Future War UI concept adds value by creating an immersive, retro-futuristic interface that enhances both gameplay and storytelling.

Problem Statement

Challenges with the Existing UI from terminator: resistance:

    <
  • Cluttered HUD and Menus: Players struggle to quickly access critical information and mission objectives.
  • Weak Thematic Integration: UI elements lack cohesion with the franchise’s dystopian, tech-driven aesthetic.
  • Poor Visual Hierarchy: Important gameplay data gets lost among less relevant information.
  • Limited Accessibility Options: The current UI fails to accommodate various player needs and preferences.

Why a Design for a hypothetical sequal?

  • To deliver a cinematic, immersive, and faction-specific interface that strengthens storytelling and worldbuilding.
  • To enhance usability with streamlined menus, tactical overlays, and clear mission tracking.
  • To modernize the interface while staying true to the gritty, war-torn aesthetic of the Terminator universe.

Research

Persona

User Insight

Alex Carter is a dedicated Halo fan and competitive gamer who loves the idea of engaging progression, a good narrative, and strategic gameplay. However, he gets annoyed by the fact that there are no mission replays, the Battle Pass is too slow, and the live service model is not finished in Halo Infinite. He wants a sequel that improves on these mechanics, making the experience more rewarding, varied, and immersive.

How Might We

  • Create a more rewarding progression system that keeps Alex engaged and motivated through skill-based unlocks and meaningful milestones.
  • Expand Halo’s open-world environments to include more biome diversity and immersive exploration, reducing repetitiveness and enhancing discovery.
  • Improve co-op and multiplayer experiences to foster a stronger sense of community through better matchmaking, shared objectives, and social features.
  • Refine storytelling and emotional stakes by deepening character development and creating meaningful narrative choices that connect players to the world.

POV Statement

As a competitive and lore-driven gamer, Alex Carter wants a Halo sequel that has better mission replayability, a rewarding progression system, and a richer, more diverse world because he wants to feel consistently challenged and engaged without the frustration of grindy or incomplete content.

UI Improvements

    Improved HUD and Accessibility:Customizable UI for better visibility and player preference.

    Enhanced Navigation: Improvements to UI navigation with a back-to-basics design.

    Dynamic Multiplayer UI: A hybrid of Halo Reach and Halo 5’s UI system while keeping the core intact.

    Store Improvement: More intuitive navigation and an overhauled UI for item display and information accessibility.

  • Customization Overhaul:
    • More intimate UI navigation, removing the expansive multiple-menu system.
    • One-to-two screen options for easier customization.
    • Ability to name your Spartan and create Spartan classes.
    • All earned and purchased items (battle pass/store) accessible in one unified area.
