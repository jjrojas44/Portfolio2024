Terminator: Future war

The UI for the terminator: resistance sequel,Terminator: Future war.

Case Study:

Terminator: Future war
Project Overview
  • Project Name: Terminator: Future war (Hypothetical Sequel)
  • Role: UX/UI Designer (Solo Designer)
  • Tools Used: Figma,Photoshop,Photopea,Gameuidatabase,dribbble,Behance,
  • Timeline: 3 Weeks
  • Objective: Introduce a hypothetical UI concept for a sequel called Terminator: Future War.
Value Added

The Terminator: Future War UI concept adds value by creating an immersive, retro-futuristic interface that enhances both gameplay and storytelling.

Problem Statement

Challenges with the Existing UI from terminator: resistance:

  • Cluttered HUD and Menus: Players struggle to quickly access critical information and mission objectives.
  • Weak Thematic Integration: UI elements lack cohesion with the franchise’s dystopian, tech-driven aesthetic.
  • Poor Visual Hierarchy: Important gameplay data gets lost among less relevant information.
  • Limited Accessibility Options: The current UI fails to accommodate various player needs and preferences.

Why a Design for a hypothetical sequal?

  • To deliver a cinematic, immersive, and faction-specific interface that strengthens storytelling and worldbuilding.
  • To enhance usability with streamlined menus, tactical overlays, and clear mission tracking.
  • To modernize the interface while staying true to the gritty, war-torn aesthetic of the Terminator universe.

Research

Persona

User Insight

  • Players found Terminator: Resistance's UI clunky, cluttered, and hard to navigate during combat sequences.
  • Many desired a more immersive, Skynet-era digital aesthetic that felt authentic to the franchise.
  • Users preferred quick-access menus, context-based HUD elements, and modular inventory systems.
  • Majority disliked the lack of mobile and console optimization in menu navigation and inventory management.

POV Statement

The user interface I want should be clean and customizable to enhance gameplay while maintaining immersion so I can stay engaged in the story and focus on what matters most — the experience.

How Might We

  • How might we design a futuristic, minimal HUD that enhances immersion during high-stakes combat?
  • How might we simplify menu structures to improve accessibility on both PC and console?
  • How might we integrate franchise-consistent UI aesthetics without sacrificing usability?
  • How might we improve inventory and mission tracking experiences with contextual, modular layouts?

Heuristic Evaluation

Purpose: Ensure UI aesthetics align with Terminator lore and Skynet visuals.

Sources: Terminator: Dark Fate, Terminator: Salvation, Terminator resistance, T-800 HUD footage.

Outcome: A franchise-consistent UI style guide for Future War.

Evaluation Summary

  • Visibility of System Status: Metro Exodus and Division 2 excel; Terminator lacks clear threat indicators.
  • Match Between System & Real World: Metro’s in-world UI is immersive; BO6 HUD too abstract at times.
  • User Control & Freedom: Division 2 leads in UI customization; Terminator is too rigid.
  • Consistency & Standards: BO6 and Division 2 follow genre conventions; Metro uses unique controls.
  • Error Prevention: Division 2 prevents gear loss; Terminator lacks save overwrite warnings.
  • Recognition vs. Recall: BO6 and Division 2 use strong iconography; Metro menus are layered.
  • Flexibility & Efficiency: Division 2 offers scalable HUD; Terminator is static.
  • Aesthetic & Minimalist Design: Metro’s HUD fades for immersion; BO6 is overly busy mid-combat.
  • Error Recovery: Division 2 gives helpful error prompts; Terminator uses generic ones.
  • Help & Documentation: Division 2 provides clear in-game tutorials; Metro lacks menu guidance.

Key Takeaways

Strengths: Division 2 excels in customization and error prevention. Metro delivers immersive story-driven UI. BO6 keeps controls familiar.

Weaknesses: Terminator lacks HUD flexibility and error transparency. BO6 HUD gets too cluttered during combat.

Opportunities: Add HUD customization in Terminator. Improve menu tutorials in Metro. Include a minimalist HUD preset in BO6. Use Division 2’s system as a UI model.
