Case Study:

Universal Studios website redesign
Project Overview
  • Project Name: Universal Studios Website Redesign
  • Role: UX/UI Designer (Solo Designer)
  • Tools Used: Figma, Adobe XD, Photoshop, Google Drive/Docs, Coolors
  • Timeline: 5 Weeks
  • Objective: Enhance usability, navigation, and booking experience while modernizing the website's design for a seamless user journey.
Value Added

The Universal Orlando website would benefit from a UX/UI redesign that creates an improved user experience through simple navigation while avoiding excessive text. The combination of visual cues and smooth interactions enables users to locate information easily, leading them through their journey without difficulty.

Problem Statement

Challenges with the Existing Website:

  • Confusing Booking Process: Users struggle to find ticket options and complete purchases.
  • Overwhelming Navigation: Too many menu items create frustration.
  • Outdated Visual Design: Poor UI hierarchy and inconsistent branding.
  • Lack of Mobile Optimization: The mobile experience is clunky and difficult to use.

Why a Redesign?

  • To provide a modern, streamlined experience that enhances usability and increases conversions.
  • To improve user satisfaction with an intuitive, visually engaging website.
  • To create a more immersive on-brand experience, aligning with Universal Studios' entertainment focus.
