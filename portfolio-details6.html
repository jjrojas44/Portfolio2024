U-Game-U-PLAY

An all You need Video game information website .

Case Study:

A True Gamers Source of Video Game Information
Project Overview
  • Project Name: Universal Studios Website Redesign
  • Role: UX/UI Designer (Solo Designer)
  • Tools Used: Figma, Figjam, Adobe XD, Photoshop, Google Drive/Docs, Coolors
  • Timeline: 4 Weeks
  • Objective: To provide a seamless gaming hub with news, reviews, deals, and a vibrant community, enhancing the player experience.
Value Added

The "u-game-u-play" case study receives value enhancement through data-driven methods which improve content creation and user engagement. The site can improve its offerings by tracking user behavior while collecting feedback about popular topics. The platform should implement A/B testing and reward users who participate in reviews or forum activities to boost engagement. The strategy would make "u-game-u-play" the primary destination for gamers.

Problem Statement

Challenges with the Existing Platform:

  • Scattered Gaming Content: Users struggle to find relevant news, reviews, and deals efficiently.
  • Poor Community Engagement: Limited interaction tools make discussions and forums less engaging.
  • Unintuitive Navigation: The layout lacks clear pathways, making discovery difficult.
  • Inconsistent Visual Identity: The design fails to create a cohesive and immersive gaming experience.

Why a Redesign?

  • To centralize gaming content, ensuring users can easily access news, reviews, and deals in one place.
  • To enhance community engagement with interactive features and a seamless forum experience.
  • To streamline navigation for better user flow and accessibility.
  • To create a visually dynamic and engaging platform that resonates with gamers.
Research & User Insights
Persona
User Insight

Steven who is a 23-year-old front end software developer from Florida has a strong preference for the latest and most advanced video games and consoles. His desire to stay informed about the latest releases and deals is due to his keen understanding of the shortcomings of current information sources in pointing out hard to find games and deals. Therefore, Steven is in search of a reliable and detailed source that can meet his needs and give him the latest news and updates on video game releases and deals.

How Might We
  • Centralize gaming content to ensure users can effortlessly access news, reviews, and deals in one place.
  • Enhance community engagement by integrating interactive discussion forums and user-driven content.
  • Improve site navigation to create a seamless, intuitive browsing experience for all users.
  • Develop a visually engaging platform that reflects the excitement and diversity of the gaming world.
